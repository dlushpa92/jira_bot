# its_bot

### Развернуть проект
#### Установка
	1. Склонировать проект: `git clone https://github.com/dlushpa92/its_bot.git`
	2. Перейти в проект: `cp its_bot`
	3. Создать файл appsettings.json(скопировать с appsettings.example.json). Указать необходимые параметры.

#### Запуск
	1. Установить Nuget пакеты (если требуется)
    2. Очистить проект: dotnet clean
	3. Перестроить проект: dotnet build
